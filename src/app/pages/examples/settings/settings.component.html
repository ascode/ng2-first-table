工具栏-设置示例

前提条件：toolData.isShow 设置为true; （工具栏必须显示）
toolData.setStyle.isShow 设置为true; （设置必须显示）
注：行拖动 / 查看明细 只允许单行操作，如果开启多行，当选中行 > 1 会自动关闭。

演示源

配置说明

    
        // 自定义工具栏
        toolData: {{ lf }}
            isShow: true, 
            // 自定义设置
            columnSetting: {{ lf }}
                // 设置是否显示
                isShow: true,
                // 设置行高
                setTrHieht: {{ lf }}
                    isShow: true,
                    setTrHiehtContent: '设置行高',
                {{ rf }},
                // 设置行拖动
                setTrMove: {{ lf }}
                    isShow: true,
                    setTrMoveContent: '选中行拖动',
                    default: 20,
                {{ rf }},
                // 查看明细
                details: {{ lf }}
                    isShow: true,
                    detailsContent: '查看明细',
                {{ rf }},
            {{ rf }},
          
        {{ rf }}
