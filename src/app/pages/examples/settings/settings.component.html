工具栏-设置示例

前提条件：toolData.isShow 设置为true; （工具栏必须显示）
toolData.setStyle.isShow 设置为true; （设置必须显示）
注：行拖动只允许单行拖动，如过开启行拖动，默认是不启用选中多行，如果业务需要可选多行，当选中行 > 1 会自动关闭行拖动。

演示源

配置说明

    
        // 自定义工具栏
        toolData: {{ lf }}
            isShow: true, 
            // 设置样式
            setStyle: {{ lf }}
                // 设置是否显示
                isShow: true,
                // 设置行高
                setTrHieht: {{ lf }}
                    isShow: true,
                    setTrHiehtContent: '设置行高',
                {{ rf }},
            {{ rf }},
          
        {{ rf }}
